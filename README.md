
# 投資部-用語集 スクレイピングツール

このプロジェクトは、複数の証券会社の用語集サイトから投資用語、読み方、および意味をスクレイピングし、Google スプレッドシートを自動更新するためのツールです。

## 特徴
- **バックグラウンド・スクレイピング**: ブラウザを開かず、サーバーサイドで高速にデータを取得します。
- **重複排除とリライト**: 複数社の情報を統合し、最適な文章にアレンジしてスプレッドシートに書き込みます。
- **Cloud Run / Cloud Build 対応**: クラウド環境での定期実行に対応。

## 対応サイト
1. SMBC日興証券
2. 岡三オンライン
3. 楽天証券
4. 大和証券
5. 三菱UFJモルガン・スタンレー証券
6. 野村證券

## 使い方（ローカル実行）
1. `pip install -r requirements.txt` を実行。
2. `service-account.json` をルートディレクトリに配置。
3. `python main.py` を実行。
